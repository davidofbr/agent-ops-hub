## Rules

1. **Never invent.** A field without a source gets `[UNKNOWN]`, not a
   plausible-sounding guess.

2. **Provenance tags per field.** Every bullet carries `[V]`, `[I]`,
   or `[H]`. Section-level markers only for sections dominated by
   non-validated facts.

3. **Marketing copy is `[I]` by default.** Testimonials and case studies
   are `[I]`. Only user interview responses and legal documents earn `[V]`.

4. **ICP is aspirational when derived from copy alone.** If Source
   diversity has no CLIENT_VOICE contribution, the confidence header
   sources score caps at 60. The file can still be `PARTIAL` or
   `INCOMPLETE`, but it will not read `COMPLETE`.

5. **No persona-level content here.** No "meet Sarah the Marketing
   Director" narratives. Keep it segment-level. Individual archetypes
   belong in `personas.md`.

6. **The confidence header is the top two comment lines of the file.**
   No content precedes them. Consumer skills parse them on load.
